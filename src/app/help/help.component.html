What you can do with HANA Server Operating System Monitoring Tool

HANA Server Operating System Monitoring Tool has been tested on Google Chrome, Microsoft Edge and Mozilla Firefox. For better experience please use Google Chrome.

You are only allowed to access this tool within SAP network via SAP computers with standard SAP windows image being installed. If the home page is empty, please ask your system administrator to add the relative locations' info to your account.

Signing in You need to login before you can access this tool

Normally, login will be done automatically. If you have multiple valid certifications, you might need to select the correct SAP certification manually.

When logging in the tool, if you are getting the error message : "User [Your ID] does not exist in the system.", please contact system administrator to add your info into the system; If you are getting the error message: "undefined error", please use SAP standard windows image to access this tool.

After logging in the tool successfully, you should see your employee id and your name at the top right corner.

Servers Status Monitoring Check CPU/Memory/Disk status
  • Checking current CPU/Memory/Disk status of all servers at Home
  • Checking the detail/history resource (CPU/Memory/Disk) consumption information of some server at Home by clicking on some server or some resource.
Administration of Users and Locations CRUD of Users and Locations
  • Managing the users and locations at Administration by clicking on the first item
  • Managing the users information at Administration by clicking on the second item
Administration of SID and Mappings CRUD of SID and Mappings
  • Managing the SID information, setting important instance flag at Administration by clicking the third item
  • Managing the SID mapping information at Administration by clicking the forth item
Administration of configuration Select/Update of Configuration

Managing/view the configuration for both backend script and web at Administration by clicking the last item
